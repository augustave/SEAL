# SEAL

**Brand doctrine system for defense-tech and American Dynamism, organized along the SEA / AIR / LAND operational-domain taxonomy.**

Five canonical palettes. Six document types. Eight iron rules.

> *Sea, air, land — every domain has a doctrine. Every doctrine has a look.*

**Live demo:** [augustave.github.io/SEAL](https://augustave.github.io/SEAL/)
**Skill download:** [`SEAL.skill`](./SEAL.skill)

---

## What this is

A Claude skill in the AKTE mold — installable, opinionated, fixed typology, non-negotiable iron rules. It generates brand doctrine for the new defense-industrial stack: Anduril, Saronic, Shield AI, Epirus, Hadrian, Hermeus, Apex, Hidden Level, Kratos, General Atomics, et al.

The name encodes the system's elasticity. SEAL is the canonical military framing of operational domains — sea, air, land — extended in this system by spectrum (Anechoic) and substrate (Rustbelt). The five palettes occupy the full operational surface area; the relationships between them are the brand.

This is not a moodboard. Each palette is constrained by 30+ years of cited industrial heritage, hardware-grade color specifications (RAL, Pantone, FED-STD 595, Cerakote), WCAG-verified contrast, and a one-line thesis that has to compress to a single sentence.

| **Air** | Apex Heraldic (`SEAL—AH`) | Above-all layer |

| **Spectrum** (across all) | Anechoic Minimal (`SEAL—AN`) | Invisible spine |
| **Industrial substrate** (under all) | Rustbelt Kinetic (`SEAL—RB`) | Production base |

Sea, Air, and Land are operational layers. Spectrum sits over all of them. Industrial sits beneath all of them. The stack relationship is the brand.

## The five palettes

### `SEAL—CW` Coldwater Pragmatism — *sea*

**Domain.** USV / UUV / AUV / glider / seafloor sensor / ASW picket / naval swarm.
**Canonical companies.** Saronic, Saildrone, Anduril Dive-LD, HII REMUS, Leidos Sea Hunter, BAE/Boeing Orca.
**Signature.** Haze grey 5-H. Anechoic hull black. Trench navy.
**Accent.** Safety orange `#FF4F00` — survival, flotation, emergency state.
**Lineage.** Bath Iron Works, Electric Boat, Royal Navy Type 26, Cold War SOSUS, North Sea oil platforms, Refn / *Vigil* / Cold War submarine cinema.
**Thesis.** *The ocean doesn't care about your DoD rating. Build like it.*

### `SEAL—RB` Rustbelt Kinetic — *industrial substrate*

**Domain.** Manufacturing / Arsenal / production base / supply chain.
**Canonical companies.** Anduril Arsenal-1, Hadrian, Firestorm Labs, Apex Space, Hermeus Quarterhorse, Saronic (Texas), Shield AI production.
**Signature.** Bridgeport machine green. Forged iron. Concrete grey.
**Accent.** MIG arc-blue `#7AB5FF` — unit-complete state, manufacture flash.
**Lineage.** Willow Run, River Rouge, Kaiser Shipyards, Detroit Arsenal, Sheeler / Bourke-White / Hine, the Arsenal of Democracy.
**Thesis.** *The prototype is the brochure. The line is the product.*

### `SEAL—AN` Anechoic Minimal — *spectrum*

**Domain.** EW / directed energy / RF / SIGINT / phased arrays / GaN ecosystem.
**Canonical companies.** Epirus Leonidas, Lockheed HELIOS, BlueHalo Locust, Anduril Pulsar, Hidden Level, L3Harris CREW, BAE EW.
**Signature.** True blackout. Mu-metal silver. Rack chassis.
**Accent.** P31 phosphor `#3CFF7A` — active signal, live trace, confirmed lock.
**Restricted secondaries.** TEMPEST red (red/black separation), Keysight cobalt (instrumentation chrome).
**Lineage.** Bell Labs, HP/Agilent/Keysight, Tektronix, Cold War SIGINT, *The Conversation*, *Tinker Tailor Soldier Spy*, the Cypherpunk-era visual record.
**Thesis.** *Spectrum is terrain. We hold the high ground.*

### `SEAL—DP` Desert Pragmatism — *land*

**Domain.** Group-1/2 UAS / austere ground / FOB infrastructure / counter-UAS / loitering munitions.
**Canonical companies.** Anduril (Sentry, Ghost, Lattice), Skydio X10D, AeroVironment Switchblade/Puma/Raven, Shield AI V-BAT, Teal, Performance Drone Works.
**Signature.** Flat Dark Earth (Cerakote H-265). Coyote brown. Bone.
**Accent.** VS-17 cerise `#E5004C` — friendly ID, ground-to-air recognition.
**Lineage.** The CENTCOM visual record from 2003 forward. Magpul. Crye Precision. Combat-camera footage. *Restrepo*, *Generation Kill*, *The Hurt Locker*.
**Thesis.** *Build for the FOB. Everything else follows.*

### `SEAL—AH` Apex Heraldic — *air*

**Domain.** Air superiority / strategic strike / hypersonics / standoff weapons / strategic ISR.
**Canonical companies.** Anduril Fury / Bolt / Roadrunner, Hermeus Quarterhorse / Darkhorse, General Atomics MQ-9 / MQ-20, Kratos XQ-58 Valkyrie, Northrop B-21, Lockheed F-35 / NGAD, Hidden Level (airspace), Shield AI V-BAT (Hivemind).
**Signature.** Old Glory blue. Air Force blue (Pantone 5395). Service white.
**Accent.** Heraldic gold `#E0AA0F` — rank, unit decoration, distinguished service.
**Restricted secondary.** Old Glory red `#BF0A30` — combat decoration, valor only.
**Lineage.** Strategic Air Command. The SR-71 Skunk Works. Plant 42 (B-2 / B-21). USAF service heraldry — ribbons, chevrons, unit emblems. AFI / Technical Order publication grammar. *Strategic Air Command* (1955), *Twelve O'Clock High*. The MITRE / RAND / JASON technical-publication aesthetic.
**Thesis.** *Range is power. Altitude is authority.*

## The eight iron rules

Non-negotiable. Every output must obey all eight. If a brief asks for something that violates a rule, the rule wins.

1. **Mass over swagger.** Defense aesthetics carry tonnage. Reject the Silicon Valley register.
2. **Anti-glint is doctrine.** Matte only. The only permitted shine is environmental.
3. **One protected accent per palette.** Semantic class — alert, signal, ID, completion. Never decoration.
4. **Real units only.** Bearings, knots, decibels, PSI, takt time. Never *blazing fast*, *next-generation*, *AI-powered*.
5. **Cultural lineage required.** 30+ year industrial heritage. A palette without lineage is a moodboard.
6. **Stack-aware.** Every palette declares its position in the doctrine stack. The stack is the brand.
7. **Reject the tourist palette.** The carrier's palette wins, never the payload's.
8. **One-line thesis, on the cover.** No palette is finished until the line is written.

Full rules in [`SKILL.md`](./SKILL.md).

## The six document types

| Code | Type | Use when |
|---|---|---|
| `SEAL—01` | **CODEX** | Default. Full ~10-section doctrine entry. |
| `SEAL—02` | **CARD** | Single-page summary. ~250 words. |
| `SEAL—03` | **STACK** | Cross-palette relationship map. |
| `SEAL—04` | **BRIEF** | Apply a palette to a specific company. |
| `SEAL—05` | **COVER** | Thesis-line + signature image direction. |
| `SEAL—06` | **INDEX** | Full taxonomy view. |

When ambiguous, default to `CODEX`.

## Install

**As a Claude skill.** Download [`SEAL.skill`](./SEAL.skill). Install via Claude's skill registry.

**Trigger phrases.** *defense aesthetic · brand doctrine · palette · visual operating language · VOL · the maritime layer · the spectrum · the FOB · range and altitude · the production base.* Or simply name a canonical palette: *Coldwater · Rustbelt · Anechoic · Desert · Apex Heraldic.*

**Output.** One of the six document types. The skill defaults to `CODEX` when ambiguous.

**Out of scope.** Civilian brand identity. Product design. Lifestyle moodboards. Use AKTE for archival/bureaucratic documents; use KAMI for editorial/literary documents.
