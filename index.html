§ 01

Framing

Codex Entry / Self-Referential

SEAL is a Claude skill in the AKTE mold — installable, opinionated, four-palette canon, six document types, eight iron rules. It generates brand doctrine for the new defense-industrial stack: Coldwater Pragmatism for the maritime layer, Rustbelt Kinetic for the production base, Anechoic Minimal for the spectrum, and Desert Pragmatism as the founding aesthetic that the others define themselves against.

The premise: defense-tech companies need doctrinally specific visual languages, not generic "futuristic" mood-boards. Each operational domain (sea / factory / spectrum / ground) has its own constraints, lineage, and accent grammar. SEAL codifies those.

This document is itself a CODEX entry — the system documenting itself in its own format. Read as a structural reference for what a finished doctrine entry looks like rendered.

§ 03

The Six Document Types

Output / Typology
SEAL—01

Codex

Full ~10-section expanded doctrine entry. The headline artifact. Default when the user says "expand," "write," or "draft."

SEAL—02

Card

Single-page summary: name, vibe, palette, three signature visuals, thesis line. ~250 words.

SEAL—03

Stack

Cross-palette relationship map. How the four interlock — payloads, supply chains, dependencies. The VOL infrastructure view.

SEAL—04

Brief

Application document. Project one palette onto a specific company, product, or campaign. Brand audits and pitch prep.

SEAL—05

Cover

Thesis-line + signature image direction. Title slides, deck covers, single-image brand statements. ~50 words plus visual brief.

SEAL—06

Index

Full taxonomy view. All four palettes with one-line summaries and pointers to the codices. Onboarding / inside-front-cover.

§ 04

The Eight Iron Rules

Non-negotiable
RULE 01

Mass over swagger

Defense aesthetics carry tonnage. Reject Silicon Valley register — agility, lightness, optimism, friendliness, "delight." The work has weight because the hardware does.

RULE 02

Anti-glint is doctrine

Matte finishes only. No gradients in UI. No beveled chrome. The only permitted shine is environmental — water, glass, metal under direct light — never finish itself.

RULE 03

One protected accent

Each palette reserves one high-saturation color as a semantic class — alert, signal, ID, completion. Never decoration. CW = orange. RB = arc-blue. AN = P31. DP = cerise.

RULE 04

Real units only

Bearings, knots, decibels, PSI, hours-on-station, units-per-month. Never "blazing fast," "next-generation," "AI-powered." If a number can be measured, use the instrument's unit.

RULE 05

Cultural lineage required

Every palette traces a 30+ year industrial heritage that predates the current company. Bath Iron Works, Bell Labs, Willow Run. A palette without lineage is a moodboard.

RULE 06

Stack-aware

Every palette declares its position in the stack — which palettes carry its payload, which feed its supply, which it depends on. No palette stands alone.

RULE 07

Reject the tourist palette

Do not borrow another palette's signifiers casually. Coldwater does not wear FDE. Anechoic does not wear safety orange. The carrier's palette wins.

RULE 08

Thesis line on the cover

Every palette ships with one quotable sentence — the line that goes on the cover when presented. No palette is finished until its line is written.
